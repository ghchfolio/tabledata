Uber Responsive Tables
Features
  • Create as many columns as you want, it'll display vertically on smaller screens
  • On smaller screens, a dropdown menu appears for sorting
  • Clicking column headers (or dropdown menu) sorts table (toggles between ascending/descending order)
  • Optionally, table cells can be place at the top of each section for mobile (e.g. 'Actions' buttons)
  • a11y friendly
