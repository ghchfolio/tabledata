Uber Responsive Tables
Features
  • Table data - no matter how many - are neatly displayed on smaller view-ports
  • On smaller view-ports, a dropdown menu appears for sorting
  • Clicking column headers (or dropdown menu) sorts the relevant column
  • Continuously clicking column headers (or dropdown menu) toggles between ascending/descending sort order)
  • Optionally, table cells (e.g. the 'Actions' buttons) can be placed at the top of each section on smaller view-ports
  • a11y friendly
